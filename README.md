# 🚀 Nexora

> A lightweight web application built with **Node.js**, **HTML**, **CSS**, and **JavaScript**.

---

## 📖 About

Nexora is a web application powered by a Node.js backend with a vanilla HTML, CSS, and JavaScript frontend.

The repository currently contains the core source code required to run the application locally.

---

## ✨ Features

- ⚡ Node.js backend
- 🎨 HTML & CSS frontend
- 📜 Vanilla JavaScript
- 📦 npm-based project
- 🚀 Lightweight project structure

---

## 🚀 Getting Started

### Prerequisites

- Node.js
- npm

### Clone the repository

```bash
git clone https://github.com/rxt1k/nexora.git
cd nexora
```

### Install dependencies

```bash
npm install
```

### Start the application

```bash
node server.js
```
